# How They Test

A collection of testing patterns and approaches used by prominent open-source developers and projects.

## Developers

- [Evan You](./evan-you/) — Creator of Vue.js, Vite, Vitest. Analysis of testing patterns across Vue 3, Vite, Vitest, Vue Router, Pinia, and VitePress.
- [Tanner Linsley](./tanner-linsley/) — Creator of TanStack. Analysis of testing patterns across TanStack Query, Router, Table, Form, and Virtual.
